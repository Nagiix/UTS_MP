# UTS_MP

Sebuah dashboard admin simpel dengan fitur:
1. Login/Logout
2. Forgot Password
3. Basic CRUD (activity dan user)
4. Basic Filters (activity - prioritas dan tipe)
5. Basic Search (activity name)
6. Dashboard page dan User page
7. Bottom navbar untuk navigasi
